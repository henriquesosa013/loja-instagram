# 🚀 Loja de Seguidores para Instagram

Uma loja desenvolvida em **Flask** para venda de seguidores, curtidas e visualizações para Instagram.

Criei este projeto porque percebi que muitas lojas de seguidores para Instagram atuais possuem um visual pouco atrativo e uma interface confusa.
Este projeto ainda está em desenvolvimento, e esta é apenas a versão base. Ainda há muitas funcionalidades e melhorias que serão adicionadas.

# Objetivos futuros

- Sistema de pagamentos
- Banco de dados mais estruturado
- Melhorias de segurança
